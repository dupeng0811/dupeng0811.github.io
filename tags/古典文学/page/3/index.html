《桃花扇》可谓中国古典戏剧的最后一部杰作，在许多方面均富有艺术创造性。文学史上有不少传奇通过男女主角的悲欢离合，串演一代兴亡的历史故事，但《桃花扇》的出现，却使这类创作达到新的艺术高度。剧中人物的构思与描写颇具匠心，剧中词曲清婉流畅，感情色彩极为浓重。
